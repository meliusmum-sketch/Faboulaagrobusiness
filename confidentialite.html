2. Données collectées

Nous collectons uniquement les données que vous nous transmettez volontairement via notre formulaire de contact :

  • Nom complet
  • Adresse email
  • Numéro de téléphone
  • Message / objet de la demande

Aucune donnée n'est collectée automatiquement, sauf les données d'usage anonymisées par notre outil d'analyse (Google Analytics), qui peuvent inclure votre adresse IP anonymisée, le type de navigateur et les pages visitées.

3. Finalité du traitement

Vos données sont utilisées exclusivement pour :

  • Répondre à vos demandes d'information ou de devis
  • Vous recontacter dans le cadre d'un éventuel partenariat
  • Améliorer notre site web (données analytiques anonymisées)

4. Conservation des données

Les données collectées via le formulaire de contact sont conservées pour une durée maximale de 24 mois à compter de votre dernière interaction avec nous. Au-delà, elles sont supprimées.

5. Partage des données

Nous ne vendons, ne louons ni ne partageons vos données personnelles avec des tiers, sauf obligation légale. Les seuls prestataires pouvant avoir accès à vos données sont notre hébergeur web (Netlify) pour le traitement du formulaire.

6. Vos droits

Conformément à la loi sénégalaise sur la protection des données personnelles (loi n° 2008-12 du 25 janvier 2008), vous disposez des droits suivants :

  • Droit d'accès : vous pouvez demander une copie de vos données
  • Droit de rectification : vous pouvez demander la correction de données inexactes
  • Droit de suppression : vous pouvez demander l'effacement de vos données
  • Droit d'opposition : vous pouvez vous opposer au traitement de vos données

Pour exercer vos droits, contactez-nous à : contact@faloboula.com

7. Cookies

Ce site utilise des cookies techniques nécessaires au bon fonctionnement du site et des cookies analytiques (Google Analytics) pour comprendre l'utilisation du site. Vous pouvez configurer votre navigateur pour refuser les cookies.

8. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données personnelles contre tout accès non autorisé, modification, divulgation ou destruction.
